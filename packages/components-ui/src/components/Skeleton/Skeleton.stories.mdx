# Skeleton

<!-- Adicionar uma descrição para o componente -->

O componente `Skeleton` substitui o loading do componente `Card` e `Table`.

## Import

```sh
  import { Skeleton } from '@drogo-ui/components';
```

## Basic

O `Skeleton` espera receber um `height` com altura e a extensão (Ex.: 100px, 0.5rem...), mas caso nada seja passado o valor padrão é de 100%.
