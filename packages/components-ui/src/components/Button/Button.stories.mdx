# Button

O componente `Button` é utilizado para permitir que o usuário possa realizar ações dentro da aplicação.

## Import

```sh
  import { Button } from '@drogo-ui/components';
```

## Variations Theme

O `Button` pode receber a prop `variant`. As variações de tema podem criar botões preenchidos, contornados ou apenas o texto.

<Canvas>
  <Story name="Thema">
    <Button m="0.5rem" variant="filled">
      Primary
    </Button>
    <Button m="0.5rem" variant="standard">
      Secondary
    </Button>
    <Button m="0.5rem" variant="outline">
      Outline
    </Button>
    <Button m="0.5rem" variant="approve">
      Approve
    </Button>
    <Button m="0.5rem" variant="danger">
      Cancel
    </Button>
  </Story>
</Canvas>

## Size Variations

O `Button` pode receber a prop `size`. As variações de tamanho podem criar botões pequenos, medios, grandes e tamanho total disponível.

<Canvas>
  <Story name="Size">
    <Button m="0.5rem" variant="standard" size="small">
      Small
    </Button>
    <Button m="0.5rem" variant="standard" size="medium">
      Medium
    </Button>
    <Button m="0.5rem" variant="standard" size="large">
      Large
    </Button>
    <Button variant="standard" size="fullSize">
      Full Size
    </Button>
  </Story>
</Canvas>

## Link

O `Button` pode receber a prop `as`. Com ela podemos substituir o elemento html `button` por outro, por exemplo o elemento `a` para criar um link.

<Canvas>
  <Story
